Privacy Notice

1. Controller

The controller in the sense of the GDPR is Jan Kaluza; address and contact details see Imprint.

2. What this site is

www.cyjan.dev is a pure information site about the open-source project CYJAN IDS. There is no sign-up, no tracking, no advertising, no cookies, no analytics.

3. Server logs (Cloudflare)

The site is delivered via the content-delivery network of Cloudflare, Inc. When a page is fetched, technically necessary data is processed:

  • IP address (truncated/anonymised by Cloudflare)
  • Date and time of the request
  • Requested URL and HTTP status code
  • Referer and user-agent (browser type / OS)

Legal basis is Art. 6 (1) (f) GDPR (legitimate interest in stable, secure operation). Cloudflare acts as a processor and may, depending on visitor location, transfer data to the United States. See Cloudflare's notice at cloudflare.com/privacypolicy.

4. Version badge via GitHub API

The hero on the home page fetches the latest release tag from the public GitHub API (api.github.com/repos/JxxKal/ids/releases/latest). GitHub, Inc., USA, sees your IP address and user-agent for this request. Only the tag name is processed and shown in your browser; nothing is stored on this side. Legal basis Art. 6 (1) (f) GDPR.

5. Self-hosted fonts

Inter and JetBrains Mono are served from this server. No data is transferred to Google Fonts, Adobe, or other third parties for typography.

6. External links

This site links to external services (notably GitHub, demo.cyjan.dev). Once you click an external link, the privacy terms of the respective provider apply; I have no control over their content or processing.

7. Cookies, analytics, advertising

None. No cookies are set, no tracking, no profiling, no re-targeting, no advertising IDs.

8. Encryption

Access is HTTPS-only with an HSTS header. Content is additionally restricted by a strict Content-Security-Policy.

9. Your rights
